## 2.5 线程模型 (Linux)

```c
/*
 * Linux多线程反应堆架构
 * 
 * Main Thread: 初始化 + 信号处理
 *     │
 *     ├── Accept Thread: 连接接受 (1个)
 *     │       └── 新连接分发到Worker
 *     │
 *     ├── Worker Thread Pool: IO处理 (CPU核数个)
 *     │       ├── epoll_wait 事件循环
 *     │       ├── 协议解析
 *     │       └── 业务处理
 *     │
 *     ├── Timer Thread: 定时任务 (1个)
 *     │       ├── 心跳检测
 *     │       ├── 超时清理
 *     │       └── 统计上报
 *     │
 *     └── Async IO Thread: 异步写入 (1个)
 *             └── 日志/存储异步落盘
```
